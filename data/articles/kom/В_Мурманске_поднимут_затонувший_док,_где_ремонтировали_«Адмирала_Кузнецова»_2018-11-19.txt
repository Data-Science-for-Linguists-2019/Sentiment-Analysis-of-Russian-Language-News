Плавучий док, затонувший в Мурманске в ходе работ на авианесущем крейсере «Адмирал Кузнецов», будет поднят, заявил журналистам глава Минпромторга Денис Мантуров. Точные сроки поднятия плавдока пока не установлены. По словам господина Мантурова, сейчас на месте ЧП работает бригада водолазов, которая вырезает кусок металла для определения степени усталости металла.
«Будет точно подниматься, потому что иначе будет нарушен режим доступа в гавань для работы 82-го завода. <…> А что касается последующего ремонта "Адмирала Кузнецова": на сроки данная проблема никак не повлияет, потому что докование планируется по контракту только летом следующего года, либо весной 2020 года»,— сообщил министр (цитата по «Интерфаксу»).
Как рассказал господин Мантуров, для решения вопроса с доком для «Адмирала Кузнецова» обсуждают несколько вариантов: «Либо сделать конструкцию по сухому доку на 10-м заводе, что фактически через залив находится, либо рассматривается вопрос по приобретению готового дока». «У нас еще есть резервные варианты по использованию сухого дока на Севастопольском заводе»,— добавил министр.
Плавучий док ПД-50 затонул в Кольском заливе 30 октября. Один из башенных кранов дока упал на вскрытую часть корабля и повредил оборудование. По предварительной версии, причиной аварии стали перебои с подачей электроэнергии. В результате ЧП погиб один человек, трое пострадали, еще один пропал без вести.
